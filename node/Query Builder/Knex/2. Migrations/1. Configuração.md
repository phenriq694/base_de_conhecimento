# Migrations

1. Configurar a pasta onde as migrations serão salvas no arquivo 'knexfile.js':
```javascript
development: {
  connection: {
    filename: './src/database/db.sqlite'
  },
  migrations: {
    directory: './src/database/migrations'
  }, 
  useNullAssDefault: true,
}
```
